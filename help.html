❓ Aide & mode d’emploi

Centre d’aide pour les actions du menu Munch • Admin.

📐 Initialiser les feuilles — initSheets()

  • Crée Participants_validés, Paiements, Prix_planches si absents.
  • Pose les en‑têtes attendus, valeurs par défaut, formats.
  • À faire après un reset ou première mise en place.

📤 Consolider & envoyer — envoyerConfirmationsConsolide()

  • Lit Réponses (validées), consolide dans Participants_validés.
  • Pose/repère les formules Montant & Total payé.
  • Envoie les emails de confirmation (bouton PayPal si Montant > 0).

🔎 Audit des formules — auditFormulasParticipants()

  • Inspecte toutes les lignes de Participants_validés.
  • Ne modifie rien, affiche un bilan.

🛠️ Réparer les formules — fixFormulasParticipants()

  • Repose les formules Montant & Total payé manquantes/altérées.
  • Remet le format 0.00.

💣 Réécrire (forcé) — forceRewriteFormulasParticipants()

  • Réécrit les formules sur toutes les lignes, même si présentes.
  • Utiliser après import CSV/collage massif.

🧹 Purges

  • Purger Participants_validés — purgeParticipantsValides() : vide les données de la feuille (conserve en‑têtes), relance l’init.
  • Purger Participants_validés + Paiements — purgeParticipantsEtPaiements() : idem sur 2 feuilles.
  • Purge complète — purgeComplet() : Participants_validés + Paiements + Réponses (⚠️ irréversible).

ℹ️ Notes

  • Total payé est une formule (SUMIF) basée sur la feuille Paiements.
  • La WebApp PayPal met à jour Paiements et le Statut Paiement, jamais la cellule “Total payé”.
  • Après modification d’en‑têtes/structure, exécuter Init puis Réparer les formules.
